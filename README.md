# Top Anime Project

This is a JavaScript evaluation project that uses the Jikan API to display top anime.

The page structure is created dynamically with JavaScript, and the styling is handled with an external CSS file.

## Current Progress

- Created the project structure
- Added the main HTML file
- Added external CSS
- Created the main page layout with JavaScript
- Connected the project to the Jikan API
- Filtered anime with a score higher than 9

## Technologies

- HTML
- CSS
- JavaScript
- Jikan API
